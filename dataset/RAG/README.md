## Getting Started

Instructions on setting up your project locally.

### Prerequisites

### Python virtual env

  ```sh
  python3 -m venv venv_rag
  source venv_rag/bin/activate
  pip install -U pip
  ```

### Installation

  ```sh
  pip install -r requirements.txt
  ```
