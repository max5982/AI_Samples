## Getting Started

Instructions on setting up your project locally.

### Prerequisites

  ```sh
  sudo apt update
  ```

### Python virtual env

  ```sh
  python3 -m venv venv_llama
  source venv_llama/bin/activate
  pip install -U pip wheel
  ```

### Installation

  ```sh
  pip install -r requirements.txt
  ```
