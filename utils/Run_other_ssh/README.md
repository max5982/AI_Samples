## Overview
To run some tasks with heavy workload from a server in pyhon code.

### Python virtual env for server

  ```sh
  python3 -m venv venv
  source venv/bin/activate
  pip install -U pip
  ```

### Python virtual env

  ```sh
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  ```
